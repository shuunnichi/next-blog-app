# カメラ許可ガイド

## カメラアクセスエラーが発生した場合

SilentEye Agent画面でカメラアクセスエラーが発生した場合、以下の手順で解決できます。

## ブラウザ別の設定方法

### Google Chrome / Microsoft Edge

1. ブラウザのアドレスバー左側の **🔒アイコン** または **カメラアイコン** をクリック
2. 「カメラ」の設定を **「許可」** に変更
3. ページを再読み込み（F5キー）または「🔄 再試行」ボタンをクリック

**または**

1. 右上の **⋮メニュー** → **設定** を開く
2. **プライバシーとセキュリティ** → **サイトの設定** を選択
3. **カメラ** をクリック
4. `localhost:3000`（または使用中のURL）を **許可されたサイト** に追加

### Safari (iOS/macOS)

**iOS (iPhone/iPad):**
1. **設定アプリ** を開く
2. **Safari** を選択
3. **カメラ** を **「許可」** に設定
4. Safariアプリに戻り、ページを再読み込み

**macOS:**
1. Safariのアドレスバー左側の **🔒アイコン** をクリック
2. **カメラの設定** を **「許可」** に変更
3. ページを再読み込み

### Firefox

1. アドレスバー左側の **🔒アイコン** をクリック
2. **接続の詳細を確認** → **詳細情報**
3. **許可設定** タブを開く
4. **カメラ** を **「許可」** に変更
5. ページを再読み込み

## よくある問題

### カメラが見つかりません
- デバイスにカメラが接続されているか確認
- 外付けカメラの場合、USBケーブルが接続されているか確認
- スマートフォンの場合、カメラアプリが正常に動作するか確認

### カメラが他のアプリで使用中
- 他のビデオ通話アプリ（Zoom、Teams、Skypeなど）を終了
- 他のブラウザタブでカメラを使用していないか確認
- デバイスを再起動

### HTTPSではない場合の制限
- カメラAPIは **HTTPS** または **localhost** でのみ動作します
- 本番環境では必ずHTTPSを使用してください
- `http://192.168.x.x`などのIPアドレスではカメラが動作しない場合があります

## セキュリティについて

- カメラの許可はサイトごとに設定されます
- 信頼できるサイトのみに許可を与えてください
- SilentEyeはローカルネットワーク内でのみ使用することを推奨します

## トラブルシューティング

それでも解決しない場合：

1. **ブラウザのキャッシュをクリア**
   - Chrome/Edge: `Ctrl+Shift+Delete` → すべてクリア
   - Safari: 設定 → Safari → 履歴とWebサイトデータを消去

2. **ブラウザを再起動**

3. **デバイスを再起動**

4. **別のブラウザで試す**

5. **開発者コンソールでエラーを確認**（F12キー）

## サポート

問題が解決しない場合は、開発者コンソール（F12）のエラーメッセージをスクリーンショットして、開発者に報告してください。
